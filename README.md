# Create spider App
## 快速启动
### 安装 spider-cli-generator
```
npm install spider-cli-generator -g
```
### 创建项目
```
// 创建 material-ui 为主框架的项目
spider init my-app
cd my-app
npm start

// 创建 ant-d 为主框架的项目
spider init my-app --ant
cd my-app
npm start
```

## 项目代码地址
